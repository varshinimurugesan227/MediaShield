# 🛡️ MediaShield V2

**Original vs Edited Image Forensic Comparator**

MediaShield V2 is a web-based tool designed to analyze and detect potential tampering in images by comparing an original image with its edited version. It provides visual difference mapping, similarity scoring, and a simple forensic report.

---

## 🚀 Features

- 🔍 Image Comparison (Original vs Edited)
- 📊 Similarity Score Calculation
- ⚠️ Tampering Detection Indicator
- 🧠 Basic Image Forensic Analysis
- 🖼️ Highlighted Difference Areas
- 📄 Auto-generated Comparison Report

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: (Node.js / Python — update accordingly)
- Image Processing: OpenCV / custom algorithms

---

## 📸 How It Works

1. Upload original image
2. Upload edited image
3. Click **Compare**
4. View:
   - Similarity percentage
   - Tampering detection result
   - Highlighted difference regions
   - Summary report
